# 3 System limits requirements

## 3.1 Current limits

System shall support configurable limits for maximum and minimal current.
The firmware shall adjust the control to be within this limits.

## 3.2 Position limits

System shall support configurable limits for maximum and minimal current.
The firmware shall adjust the control to be within this limits.

## 3.3 Velocity limits

System shall support configurable limits for maximum and minimal current.
The firmware shall adjust the control to be within this limits.
